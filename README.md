# Monte-Carlo-Simulation-Of-Radioactive-Decay
Monte Carlo simulation of radioactive decay demonstrating how random processes produce predictable behavior at large scales.
One of the most interesting observations:

For large systems (N = 5000), the decay curve is smooth
For small systems (N = 50), the curve becomes noisy and jagged

Since In the real world, a radioactive atom doesn't have an "alarm clock." There is no internal countdown telling it when to decay. According to quantum mechanics, it is truly random. You can’t predict when one atom will pop, even if you have all the data in the universe.
But while I was working on this, a staggering thought came into my mind,  How a deterministic tool like Python has the "right" to simulate a non-deterministic event.

Yet, this "Pseudo" randomness is so high-quality that it perfectly mimics the laws of physics.
